## 1. Описание проекта.
opencv-image-processor** — это десктопное приложение с 
графическим интерфейсом для обработки изображений, 
разработанное на языке Python с использованием библиотек OpenCV, 
Pillow и Tkinter. Приложение предназначено для выполнения 
базовых операций по обработке изображений:
- Визуализация отдельных цветовых каналов (RGB)
- Изменение геометрических размеров
- Коррекция яркости 
- Нанесение геометрических фигур
- Захват изображений с веб-камеры
- Загрузка изображений с устройства

## 4. Установка и запуск (кросс-платформенный).
Операции выполняются в командной строке (терминале)
### Шаг 1: Клонирование репозитория
```cmd	
git clone https://github.com/AnastasiiaMay24/opencv-image-processor.git
cd opencv-image-processor
```

### Шаг 2: Создание и активация виртуального окружения
Для Windows (CMD / PowerShell)
```cmd	
python -m venv venv
venv\Scripts\activate
```
Для Linux / MacOS
```bash	
python3 -m venv venv
source venv/bin/activate
```
Примечание: 
После активации в начале строки терминала появится (venv), 
что означает, что окружение активно.

### Шаг 3: Установка зависимостей
Команда одинакова для всех ОС:
```cmd	
pip install -r requirements.txt
```
Или создайте вручную 
```cmd
pip install opencv-python pillow numpy
```

### Шаг 4: Запуск приложения
Для Windows (CMD / PowerShell)
```cmd	
python main.py
```
Для Linux / MacOS
```bash	
python3 main.py
```
После выполнения команды откроется графическое окно 
приложения Image Processor.

## 4. Руководство пользователя.
### Шаг 1. Загрузка изображения/Снимок с веб-камеры
Загрузка изображения:
1. Нажмите кнопку **"Выбрать файл"** в разделе 
"1. Загрузка изображения"
2. В открывшемся диалоговом окне выберите изображение
3. Нажмите **"Открыть"**
**Поддерживаемые форматы: PNG, JPG, JPEG**
**Результат:**
- Изображение отобразится в правой части окна
- Станут доступными все функции обработки
----------
**Пример выполнения операции:**
Действие: Выбран файл "photo.jpg"
Результат: В строке статуса появилось "Загружено: photo.jpg"
Изображение отобразилось на холсте
----------

Снимок с веб-камеры:
1. Нажмите кнопку **"Сделать снимок"** в разделе "2. Веб-камера"
2. Откроется окно камеры
3. Нажмите **ПРОБЕЛ** (SPACE) для захвата снимка
4. Нажмите **ESC** для отмены
**Результат:**
- Снимок сохраняется в папку captures
- Изображение появляется в программе
----------
**Пример выполнения операции:**
Действие: Нажат ПРОБЕЛ в окне камеры
Результат: Снимок сохранён как "capture_20260618_143025.png"
В строке статуса: "Снимок сохранен: captures/capture_20260618_0001.png"
----------


### Шаг 2. Выбор цветового канала:
Выберите один из вариантов в разделе "3. Цветовой канал":
- **Оригинал** — полноцветное (исходное) изображение
- **Красный** — только красный канал
- **Зеленый** — только зеленый канал
- **Синий** — только синий канал
**Результат:**
Изображение обновляется с выбранным цветовым каналом.
----------
**Пример выполнения операции:**
Действие: Выбран "Красный"
Результат: Изображение стало черно-белым с оттенками красного
(отображаются только значения красного канала)
----------


### Шаг 3. Изменение размера изображения:
1. В поле **"Ширина"** введите желаемую ширину (в пикселях)
2. В поле **"Высота"** введите желаемую высоту (в пикселях)
3. Нажмите кнопку **"Применить"**
**Допустимые значения:** от 1 до 5000 пикселей
**Результат:**
Изображение масштабируется до указанных размеров.
----------
**Пример выполнения операции:**
Действие: Ширина = 300, Высота = 200, нажат "Применить"
Результат: Изображение уменьшено до 300×200 пикселей
В строке статуса: "Размер изменен: 300 x 200"
----------


### Шаг 4. Понижение яркости выбранного изображения:
1. В поле **"Значение (0-255)"** введите число
2. Нажмите кнопку **"Понизить"**
**Допустимые значения:** от 0 до 255
**Результат:**
Яркость изображения уменьшается на указанное значение.
----------
**Пример выполнения операции:**
Действие: Введено значение 50, нажат "Понизить"
Результат: Изображение стало темнее на 50 единиц яркости
В строке статуса: "Яркость понижена на 50"
----------


### Шаг 5. Рисование прямоугольника синего цвета:
1. В поле **X1** введите отступ слева для левого верхнего угла
2. В поле **Y1** введите отступ сверху для левого верхнего угла
3. В поле **X2** введите отступ слева для правого нижнего угла
4. В поле **Y2** введите отступ сверху для правого нижнего угла
5. Нажмите кнопку **"Нарисовать прямоугольник"**
**Условия:**
- X2 > X1 (левый верхний угол должен быть левее правого нижнего)
- Y2 > Y1 (левый верхний угол должен быть выше правого нижнего)
Яркость изображения уменьшается на указанное значение.
- **Результат:**
На изображении появляется синий залитый прямоугольник.
----------
**Пример выполнения операции:**
Действие: X1=50, Y1=50, X2=250, Y2=200, 
нажать "Нарисовать прямоугольник"
Результат: На изображении появился синий прямоугольник
размером 200×150 пикселей
В строке статуса: "Прямоугольник: (50,50) → (250,200), размер: 200×150"
----------


### Шаг 6. Сброс к исходному изображению:
Нажмите кнопку **"Вернуть исходное изображение"**
**Результат:**
- Изображение возвращается к исходному виду
- Все изменения (размер, яркость, прямоугольник) отменяются
- Поля ввода очищаются
----------
**Пример выполнения операции:**
Действие: После всех изменений нажат "Вернуть исходное изображение"
Результат: Изображение вернулось к исходному виду
Все поля ввода очищены
В строке статуса: "Изображение возвращено к исходному"
Выбран канал "Оригинал"
----------

В нижней части окна отображается **строка статуса**, 
которая информирует о текущем действии.
Снимки с камеры сохраняются автоматически в папку captures.

### Возможные ошибки и их устранение.
В процессе работы с приложением могут возникать следующие ошибки:
1. «Нет изображения», это означает, что вы пытаетесь выполнить 
операцию (изменение размера, яркости или рисование прямоугольника) 
без загруженного изображения. В этом случае необходимо нажать 
кнопку «Выбрать файл» и загрузить изображение.
2. При попытке загрузить файл неподдерживаемого формата 
(например, BMP, GIF, TIFF) появится сообщение 
«Поддерживаются форматы: PNG и JPG». 
Решение — выбрать файл с расширением .png, .jpg или .jpeg.
3. Если в поля ввода ширины или высоты введено 
некорректное значение (ноль, отрицательное число, буквы 
или число больше 5000), программа выдаст ошибку 
«Введите корректные размеры (1-5000)». 
Необходимо ввести целое число в диапазоне от 1 до 5000.
4. При вводе некорректного значения в поле яркости 
(отрицательное число, буквы или число больше 255) 
появляется сообщение «Введите число от 0 до 255». 
Требуется ввести целое число в диапазоне от 0 до 255.
5. Ошибка «X2 должен быть больше X1, а Y2 больше Y1» возникает, 
когда координаты прямоугольника заданы неверно — 
левый верхний угол не находится левее и выше правого нижнего. 
Для исправления убедитесь, что X2 > X1 и Y2 > Y1.
6. Если прямоугольник выходит за границы изображения, 
программа сообщит «Координаты выходят за границы». 
В этом случае необходимо уменьшить координаты так, 
чтобы прямоугольник полностью помещался внутри изображения.
7. При вводе букв или дробных чисел в поля координат прямоугольника 
появляется ошибка «Введите целые числа». Следует ввести целые числа.
8. Сообщение «Не удалось загрузить изображение» означает, 
что файл поврежден или не является изображением. 
Решение — проверить файл и открыть его в другом просмотрщике.
9. Если веб-камера не обнаружена или занята другим приложением, 
появляется ошибка «Ошибка камеры». В этом случае рекомендуется 
проверить подключение камеры, закрыть другие программы 
(Zoom, Skype) и перезагрузить компьютер.
10. Если веб-камера не подключена или отсутствует, программа выдаст 
сообщение «Камера не найдена». Следует подключить веб-камеру
или использовать функцию загрузки изображения с диска.
